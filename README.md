# Projeto CRUD

## Tecnologias usadas

 - Mongoose
 - MongoDB
 - Node.js
 - Express
 - Swagger

## Iniciando o projeto

 - Primeiro baixe as dependências do projeto.

```bash
npm install 
#or
yarn
```
 - Depois inicie o projeto

```bash
npm run dev 
#or
yarn dev
```

## Rotas

Na rota /docs se encontra toda a documentação das rotas criado pelo Swagger.
